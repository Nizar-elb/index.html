Aujourd'hui, je vais vous montrer comment créer une page web grâce au langage HTML

(NB: Ceci n'est pas une chose facile)

Le HTML (HyperText Markup Language) est un langage de balisage utilisé pour créer et structurer les pages web.

Il permet d'organiser le contenu d'une page sur Internet, par exemple:

  • Les titres
  • Les paragraphes
  • Les images
  • Les liens
  • Les tableaux
