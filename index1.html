(NB:Ceci n'est une chose facile)

Le HTML (HyperText Markup Language) est un langage de balisage utilisé pour créer et structurer les page web.

Il permet d'organiser le contenu d'une page sur Internet , par exemple:

  • Les titres
  • Les pargraphes
  • Les images
  • Les liens
  • Les tableaux
